"# nightcat-Linebot" 
====
## 介紹:
---

    這是一個用Linebot做的一個小應用
	  這個專案叫做nightcat-linebot

    機器人名子叫做TODO，用來讓我存一些常用的學校網站，以及儲存一些待辦事項。

## 功能:
--
	### 1.新增待辦

	### 2.刪除待辦

	### 3.顯示待辦

	### 4.圖形化介面

## 操作:
--
	### 1.新增:
	**	在聊天室輸入 => 新增(月日)(內容) **

		Ex: 新增0522今天要去倒垃圾
	
	### 2.刪除:
	**	在聊天室輸入 => 刪除(月日)第(數字)個待辦 **

		Ex: 刪除0526第5個待辦

	### 3.顯示:
	**	在聊天室輸入 => 顯示(月日) **

		Ex:顯示0522

	### 4.圖形化介面:

	**	在聊天室輸入 => Help **
		
		=> 點選想要的內容
